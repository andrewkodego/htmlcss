My Portfolio

MyDiversify

MyDiversify is Diversify's internal HRIS system that is used by all employees and its client. It's main features are attendance tracker, employee leave balance tracker, employee payroll, client invoicing, ticketing system, asset management, inventory, purchase order, payment request form, rewards portal, and reports.

Diversify Careers

The careers web application is Diversify's own recruitment portal. Its main features are managed job posting, candidate application and processing, and online exam. This uses an API to connect and pull some data from the MyDiversify system.

Diversify Website

The company website of Diversify. Developed with the use of HTML, CSS, Bootstrap, and jQuery on the front-end, PHP and WordPress framework on the backend, and connected to a MySQL database.
